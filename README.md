字符类型本质探讨
1)字符型存储到计算机中,需要将字符对应的码值(整数)找出来
    存储：字符--->对应码值---->二进制-->存储
    读取：二进制---->码值--->字符-->读取
    
2)字符和码值的对应关系是通过字符编码表决定的(是规定好)
Go语言的编码都统一成了utf-8,和其它的编程语言来说。非常的方便，很统一，在也有编码的困扰了
